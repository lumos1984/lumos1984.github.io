4.2-this关键字（指向

  • 每一个函数内部都有一个关键字是 this

  • 函数内部的 this 指向谁，取决于函数的调用方式

this指向5种

1.普通函数 this=> window对象(浏览器核心对象) function fn() {
console.log(‘this’, this)
this.a = 10 //给window创建一个名为a的属性
}
fn()
2.对象方法中的 this=> 方法调用者 var obj = {
username: ‘admin’,
age: 23,
say: function() {
console.log(‘say this->’, this)
}
}
obj.this()
3.定时器 setTimeout setInterval this=> window对象 setTimeout function(){
console.log(‘定时器 this->’,this)
},1000)
4.事件处理函数 =>场景源 var divEle = document.querySelector(‘div’)
divEle.addEventListener(‘click’，function(){
console.log(‘事件处理函数 this->’,this)
})
5.自调用(执行函数) =>window function fn() {
console.log(‘this’, this)
this.a = 10 //给window创建一个名为a的属性
}
fn()

改变函数指向3种

1)call 函数 =>函数调用同时 改变函数中this指向对象

        + 语法 `函数名.call(要指向的对象，参数1，参数2)`

2） apply 方函数调用同时 改变函数中this指向对象

        + 语法 `函数名.apply(要指向的对象,[参数1,参数2..])`

3）bind 不会立即执行函数，返回改变this指向的新函数

+ 语法：`var newFn = 函数名.bind(要指向的对象)`

  • 不能变量提升 没有定义直接使用
  • let 和 const 声明的变量只能在块作用域起作用
